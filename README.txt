By: Chris Rose

A game about rats.

Controls:
===========================================================
W,A,S,D   = Movement
Arrows    = Attack
SHIFT     = Hold to run
SPACE     = Jump
C         = Throw coins (throw at vendors to buy stuff)
E         = Pick up / Throw items
R         = Equip item
M         = Toggle map
I         = Toggle inventory screen
U         = Toggle stats screen
Q         = Drop an item in your inventory
TAB       = Move selection down in inventory
SHIFT+TAB = Move selection up in inventory
